# Evidence Dashboard

## Setup

1. Copy the sample environment file:
   ```bash
   cp .env.sample .env
   ```
2. Open `.env` and fill in the environment variables.
   > Note: This is optional, but required to generate sources.

## Install

Install the dependencies:

```bash
npm install
```

## Dev

1. Generate sources (fetch data):
   ```bash
   npm run sources
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
